# KnockIT
*Minecraft Version: 1.8*  
**KnockIT** is a fast-paced Minecraft minigame plugin where players battle on a small platform using knockback sticks. The goal is simple: **stay on the platform as long as possible while knocking others off.**

## Gameplay  

- Players spawn on a small platform  
- Each player receives a knockback stick  
- Hit other players to knock them off the platform  
- Avoid falling yourself  

## Installation  

1. Download the plugin `knockit.jar` file  
2. Place it in your server's `plugins` folder  
3. Restart or reload your server  
4. Done!
